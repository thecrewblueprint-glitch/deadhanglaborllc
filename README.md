# Deadhang Labor LLC Website

Static public portfolio website for Deadhang Labor LLC.

## Contents

- Static HTML pages
- Global CSS
- Small JavaScript navigation helper
- Public portfolio and brand images
- Legal/privacy pages
- Sitemap, robots file, web manifest, and favicon

## Production Notes

This repository is intended for public website code only.

Do not commit:

- cPanel account files
- mail files
- logs
- database dumps
- environment files
- backups
- private documents
- contractor records
- uploaded client files
- admin dashboard code
- API keys or credentials

Future contractor tools, document uploads, dashboards, or client-facing portals should live in separate private repositories.
